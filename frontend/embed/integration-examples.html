Immigration AI - Embeddable Components Integration Guide

1. Basic Integration (Login Button + Chat Widget)

Add both the login button and chat widget to your website with minimal configuration:

<!-- Add this before closing </body> tag --> <script> // Configuration for your agency window.ImmigrationAIConfig = { agencyId: 'your-agency-id', portalUrl: 'https://immigration-ai-portal.netlify.app' }; window.ImmigrationAIChatConfig = { agencyId: 'your-agency-id', apiUrl: 'https://immigration-ai-portal.netlify.app/api' }; </script> <!-- Load the embeddable components --> <script src="https://immigration-ai-portal.netlify.app/embed/embed-login.js"></script> <script src="https://immigration-ai-portal.netlify.app/embed/embed-chat.js"></script>

2. Advanced Configuration

Customize the appearance and behavior of both components:

<script> // Login Button Configuration window.ImmigrationAIConfig = { agencyId: 'your-agency-id', portalUrl: 'https://immigration-ai-portal.netlify.app', buttonText: 'Client Login', buttonPosition: 'top-right', // top-right, top-left, bottom-right, bottom-left theme: 'light', // light or dark redirectAfterLogin: 'https://yoursite.com/welcome' }; // Chat Widget Configuration window.ImmigrationAIChatConfig = { agencyId: 'your-agency-id', apiUrl: 'https://immigration-ai-portal.netlify.app/api', position: 'bottom-right', // bottom-right, bottom-left, top-right, top-left theme: 'light', welcomeMessage: 'Welcome to [Your Agency]! How can I help you today?', enableHumanHandoff: true, businessHours: { enabled: true, timezone: 'America/New_York', schedule: { monday: { start: '09:00', end: '17:00' }, tuesday: { start: '09:00', end: '17:00' }, wednesday: { start: '09:00', end: '17:00' }, thursday: { start: '09:00', end: '17:00' }, friday: { start: '09:00', end: '17:00' }, saturday: null, sunday: null } } }; </script> <script src="https://immigration-ai-portal.netlify.app/embed/embed-login.js"></script> <script src="https://immigration-ai-portal.netlify.app/embed/embed-chat.js"></script>

3. Chat Widget Only

If you only want the chat widget without the login button:

<script> window.ImmigrationAIChatConfig = { agencyId: 'your-agency-id', welcomeMessage: 'Hi! I can help answer your immigration questions.', position: 'bottom-right' }; </script> <script src="https://immigration-ai-portal.netlify.app/embed/embed-chat.js"></script>

4. Login Button Only

If you only want the login button without the chat widget:

<script> window.ImmigrationAIConfig = { agencyId: 'your-agency-id', buttonText: 'Client Portal Access', buttonPosition: 'top-right' }; </script> <script src="https://immigration-ai-portal.netlify.app/embed/embed-login.js"></script>

Configuration Options

Login Button Options (ImmigrationAIConfig)

agencyId (required): Your unique agency identifier
portalUrl: URL to the client portal (default: our hosted portal)
buttonText: Text displayed on the button (default: "Client Portal")
buttonPosition: Position of the button - top-right, top-left, bottom-right, bottom-left
theme: Visual theme - light or dark
redirectAfterLogin: URL to redirect to after successful login

Chat Widget Options (ImmigrationAIChatConfig)

agencyId (required): Your unique agency identifier
position: Widget position - bottom-right, bottom-left, top-right, top-left
welcomeMessage: First message shown to users
enableHumanHandoff: Allow users to request human support
businessHours: Configure when human support is available

5. JavaScript API

Control the components programmatically:

// Open the client portal programmatically window.ImmigrationAI.openPortal(); // Close the client portal window.ImmigrationAI.closePortal(); // Toggle the chat widget window.ImmigrationAIChat.toggle(); // Send a message programmatically window.ImmigrationAIChat.sendMessage('Hello from the website!', false); // Show notification on chat button window.ImmigrationAIChat.showNotification();
